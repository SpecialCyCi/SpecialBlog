SpecialBlog
===========

The source code of [www.specyci.com](http://www.specyci.com).

A Rails 4 blog system, using Mongod and Redis.

## Demo

see [www.specyci.com](http://www.specyci.com)

## Requirements

* Rails 4
* Mongod

## First Run

* install mongod
* run ```rake db:seed```

## Deploy

```
$ cap deploy
```
